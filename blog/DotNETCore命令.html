# DotNETCore 命令

新建控制台项目

dotnet new console

运行控制台项目

dotnet build

  • 多了 bin 文件夹
  • obj 多了 Debug 文件夹

新建类库项目

dotnet new classlib

类库打包 dotnet restore

打包前 dotnet-demo.csproj

<Project Sdk="Microsoft.NET.Sdk">

  <PropertyGroup>
    <TargetFramework>netstandard2.0</TargetFramework>
    <RootNamespace>dotnet_demo</RootNamespace>
  </PropertyGroup>

</Project>

打包

dotnet pack 报错 System.Reflection.AssemblyCompanyAttri bute”特性重复

加入 false

打包成功
