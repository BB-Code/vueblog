iocnfont 图标库的使用

  1. 现在 阿里图库下载代码包 复制 iconfont.ttf 到新建的 assets 的 fonts 下
  2. 在 pubspec.yaml 添加
fonts:
  - family: iconfont
    fonts:
      - asset: assets/fonts/iconfont.ttf
  1. 使用前查找下载代码包 index.html 里面图标的 unicode,去掉 &#，改成 0
Icon(
    IconData(0xe63e, fontFamily: 'iconfont'),
    color: Colors.pinkAccent,
),

APP 国际化

  1. 在 pubspec.yaml 添加
dependencies:
  flutter_localizations:
    sdk: flutter

#末尾添加
flutter_intl:
  enabled: true

使用插件 Flutter Intl: Initialize 初始化项目 使用 Flutter Intl: Add locale 添加语言（'en'/'zh'...） 生成 generated 和 l10n 目录 在 l10n 对应的语言.arb 添加词语键值对

localizationsDelegates: [
        S.delegate,
        GlobalMaterialLocalizations.delegate,
        GlobalWidgetsLocalizations.delegate,
        GlobalCupertinoLocalizations.delegate,
      ],
      supportedLocales: S.delegate.supportedLocales,
      locale: Locale('zh'), //切换语言
