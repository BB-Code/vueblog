# npm 常用命令

  • npm root -g 查看包安装地址
  • npm list -g 查看全局安装的软件包
  • npm list --depth=0 仅获取顶层的软件包
  • npm list 包 指定名称来获取特定软件包
  • npm view 包 version 查看软件包在 npm 仓库上最新的可用版本
  • npm search 搜索软件包
  • npm view 包 versions 列出软件包所有的以前的版本
  • npm install 包@1.0.0 @语法来安装 npm 软件包的旧版本

# Node.js 包运行器 npx

从 npm 的 5.2 版本（发布于 2017 年 7 月）开始可用

如果不想安装 npm，则可以安装 npx (opens new window) 为独立的软件包

作用

  • 不需要安装任何东西

npx cowsay "你好"

  • 可以使用 @version 语法运行同一命令的不同版本
  • 避免使用 nvm 之类的工具或其他 Node.js 版本管理工具

npx node@10 -v

  • 当被下载完，则下载的代码会被擦除
  • 直接从 URL 运行任意代码片段

npx https://gist.github.com/xxx

# 报错处理

Warning: Setting the NODE_TLS_REJECT_UNAUTHORIZED environment variable to '0' makes TLS connections

npm config set strict-ssl false set NODE_TLS_REJECT_UNAUTHORIZED=0

# 开发环境设置

运行 npm 设置

NODE_ENV=production node app.js

获取设置

process.env.NODE_ENV //development or production
